CSS
1.Cascading style sheet
    层叠样式表,修饰和美化页面元素,实现页面的排版和布局.
2.CSS使用方式
    1.行内样式/内联样式
        语法:通过标签属性style,为元素设置css样式
        注:CSS样式声明是由CSS的属性和值组成的
        例:
            <h1 style="CSS属性名:值;"></h1>
        常用CSS属性:
            1.文本颜色 color 取颜色值
            2.背景颜色 background-color 取颜色值
            3.字体大小 font-size 取像素值
            4.text-decoration 取消a标签的下划线
    2.文档内嵌
        语法:使用<style></style>标签引入CSS代码,实现样式与结构的分离
        <style>
            CSS选择器{
                属性:值;
                属性:值;
            }
        </style>
        <h1></h1>
        <p></p>

        注:
            1.CSS选择器主要用来匹配页面中的原色为其设置样式
            2.<style></style>标签可以书写在文档的任意位置,书写任意次,一般遵循'样式优先'的原则
            3.标签选择器:通过标签名到页面中匹配元素;
    3.外链方式
        1.在HTML文件中使用外部的样式表文件,CSS文件以.css为后缀,文件中直接使用选择器书写样式,在HTML文件中使用外部的样式表文件
            中使用<link>标签引入
        2.外链方式真正实现结构与样式的分离,实现样式的复用
3.样式表特征:
    1.层叠性
        不同的样式在同一个元素上共同起作用
    2.继承性
        CSS某些样式可以被子元素或后代元素继承
        例: 大部分的文本样式;
            块元素默认与父元素宽度保持一致
    3.样式表的优先级
        元素发生样式冲突最终选用哪一种样式,由样式的优先级决定
        1. 行内样式优先级最高 (style属性定义的样式)
        2. 文档内嵌(<sytle>)和外链(link)的样式优先级 低于行内样式,发生样式冲突时,
            考虑代码的书写顺序,后来者居上(平级的优先级比较规则:后来者居上!!)
        
4. CSS选择器    
    1.作用:选择页面中那些元素应用样式
    2.分类:
        1.标签选择器/元素选择器
            1).作用:
                根据指定的标签名匹配文档中所有的该元素
            2).语法:
                标签名{
                    属性:值;
                    属性:值;
                }
            3).练习:
                创建文件,添加超链接标签
                设置超链接文本色为红色,取消下划线
                text-decoration
                示例见:demo1.html
        
        2.id 选择器
            1.作用:根据元素的id属性值选择元素
            2.id属性具有唯一性,不能重复
            3.id属性值命名规范:
                自定义,尽量见名知值;
                可以由 数字,字母,下划线, - 组成,禁止由数字开头
            4.语法:
                id选择器以#开头,后跟id属性值,用于匹配文档中唯一的元素
                例:
                    #id选择器{
                        样式声明
                    }
            5.练习
                示例见 demo2.html
        
        3. class选择器/类选择器
            1.作用:根据标签的class属性值匹配元素
            2.特点:类选择器定义的样式可以实现复用,类名可在多个元素中重复使用,命名规范参照id属性值
            3.语法:
                类选择器以英文.开头,后跟class属性值

            4.练习
            5.特殊用法:
                1.标签选择器与类选择器组合使用,注意标签名在前,类名在后
                    例:  div.c1{

                    }
                    匹配类名为c1的div元素
                        .c1.c2类选择器的组合使用
                2.class属性只能写一个,属性值可以取多个,属性值之间使用空格隔开,与类选择器一一对应,应用样式
                    例:
                        <div class='c1 c2'></div>
        
        4.群组选择器:
            1.作用:为一组元素统一设置样式
            2.语法: 
                选择1,选择2,选择3...{

                }
        
        5.后代选择器(祖祖孙孙选择器)  示例见child_select.html
            1.语法:
                选择器1 选择器2{
                    
                }
            2.做选择器1对应的元素下,查找所有满足选择器2的后代元素

        6.子代选择器(儿子选择器)
            1.语法:
                选择器1>选择器2{

                }  

            2.查找***直接***子元素
        
        7.伪类选择器
            1.作用:针对元素的不同状态设置状态
            2.分类: 
                1.超链接伪类选择器:针对超链接访问前和访问后两种状态分别设置样式
                    访问前  :link
                    访问后  :visited
                2.动态伪类选择器
                    :hover  鼠标滑过或者悬停时状态
                    :active 表示激活(鼠标点按不抬起)状态
                    :focus  文本输入框获取焦点时的状态(编辑状态)

                3.使用:
                    伪类选择器必须与其他选择器结合使用,表示某种状态
                    例:
                        a:link{

                        }
                        示例: demo3.html
                4.如果需要为超链接添加四种状态的样式,需要固定书写顺序:
                    LoVe/HAte 爱恨原则
                    :link
                    :visited
                    :hover
                    :active

    3.选择器的优先值
        在使用选择器为元素设置样式,发生冲突时,最终取决于选用哪一个选择器的样式,取决于选择器的权重

        权重: 权值,使用数字来标明选择器的优先级,数字越大,优先级越高



        选择器                权重
        标签选择器             1
        类/伪选择器            10
        id选择器              100
        行内样式优先级最高     1000
        注:
            1.组合选择器(后代,子代,伪类),权重为各个选择器权值之和
            2.群组选择器权重不累加,单独看每个选择器的权重
            示例见: demo4_select_YXJ.html

        练习:
            创建div id为nav,表示导航栏
            div中嵌套若干a标签,表示导航项.
            div:宽高背景色
            a:取消下划线,修改默认颜色,鼠标悬停修改a标签的文本色和背景色
            示例见:demo5_nac_exer.html

    
5.HTML 标签分类:
    1.块元素:
        独占一行,不与其他元素工行;可以设置宽高;注:除table外,所有的快元素默认宽度与父元素保持一致,高度有内容决定
        例: div p h1 table ul ol form li 
    2.行内元素:
        可以与其他元素共行显示,不能手动调整宽高
        例: a span label i b strong u s sub sup
    3.行内块
        既可以共行显示又可以设置宽高
        例:img  input  button
        注:
            行内块元素是特殊的行内元素,默认垂直方向上以文本底部对齐显示(基线对齐),可以通过属性 vertical-align调整行内元素的垂直对齐方式,冲用于调整行内块元素垂直对齐问题
            取值: top/middle/bottom
            baseline(默认值)

1.web
     网页,基于B/S模式的应用程序
     B/S:Browser / Server
     浏览器与服务器交互模式
     C/S: Client / Server 客户端与服务器交互模式
     
2.网页组成:
    服务器:提供数据支持
    浏览器:(1)代替用户向服务器发请求 (2)解析数据
    通信协议:规范数据传输及打包的方式
    http: Hyper Text Transfer Protocal (超文本传输协议)

3.Web服务器:
    1.作用:
        (1) 提供数据支持
        (2) 接收请求并相应
        (3) 提供数据安全功能
    2.产品:
        1.Apche
        2.Nginx
        3.Tomcat
        4.IIS (Internet Information Service)
            ...
    3.开发语言:
        1.Java
        2.PHP
        3.ASP.net
        4.Python(Django,Flask)
    
4.浏览器
    1.作用:
        (1)代替用户向服务器发送请求
        (2)接收响应数据并解析,以图形化界面的形式展示给用户
    2.主流产品:
        Chrome -- Google
        Safari -- APPLE
        IE     -- Microsoft
        Firefox-- Mozilla
        Opera  -- Opera
        划分依据:
        1.市场份额(决定每年的排名)
        2.浏览器引擎和内核的不同
            渲染引擎:渲染页面,控制渲染结果
            JS引擎:解析脚本
    3.技术:
        前端开发 -- 制作网页
        HTML:写页面结构
        CSS:写样式
        JS:写逻辑,实现交互
        框架:VUE Agular Nodejs(JS的运行环境)

5.HTML 概述
    1. HTML (Hyper Text Markup Language) 超文本标记语言
        作用:通过标签的形式书写网页结构并且填充内容

    2. 标记
        也叫标签或元素,标记页面结构和内容,配合CSS实现页面整体布局和美化

    3. 网页在计算机中以.html或 .htm 后缀标识
        网页文件的打开工具 -- 浏览器
        开发工具:记事本,sublime,vscode,EditPlus
        调试工具:Chrome 浏览器自带开发者工具,使用右键-'检查'打开,或者F12
    4.标签分类:
        单标签  双标签 
        标签嵌套:
            在双标签中嵌套添加其他标签
    5.文档结构:
        1.网页文件以.html / .htm 为后缀
        2.添加双标签<html> <html/>
        3.在<html>标签中嵌套一对<head><head/>,嵌套<body><body/>标签
    
    6.标签学习
        <link> 标签用于引入外部资源
            标签属性:
                rel 设置资源类型
                href 设置资源路径
                type 声明文件类型和后缀
    网页主题内容:
        1.文本标签
            1.标题标签
            <hn><hn/>  'h1~h6'依次减小
        2.段落标签
            <p><p/>
        3.普通文本标签
            给一段文本中的某一部分额外添加样式或者动态操作使用的标签
            (1)行分区 <span></span>
                常与段落或者标题结合使用,为内容设置额外样式
            (2) <lable></lable>
                在表单中使用,优化用户操作
            (3) 格式标签
                加粗: <b></b>或<strong></strong>  'H5推荐使用更有语义的标签'
                斜体: <i></i>
                删除线:<s></s> 或 <del></del>
                上标:<sup><sup/>
                下标:<sub></sub>





        3.字符实体:
                1. < : &lt;
                2. > : &gt;
                3. 空格 : &nbsp
                4. 版权符号 : &copy
                5. 人民币符号 : &yen;
        4.块分区标签
            容器标签,划分网页模块
            <div></div>
    4.列表标签
        1.列表是一种结构化标签,按照从上到下依次显示数据,每一条数据称为一个列表项,
          不同类型的列表,项目符号不同
        2.分类:
            1.有序列表
                默认按照阿拉伯数字从1开始排列
                语法:
                    <ol>
                        <li>List ITem</li>
                    </ol>
            2.无需列表
                <ul>
                    <li>列表项</li>
                </ul>
            默认以实心原点标识列表项
            练习:新建文件,在body中添加列表
                无序列表显示四大名著
                有序列表列表编程语言

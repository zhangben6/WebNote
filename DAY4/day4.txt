1.尺寸与颜色取值
    1.尺寸单位
        1. px : 浏览器默认单位,表示像素
        2. % : 百分比单位,如果当前元素自身有尺寸(图片),百分比单位参照自身尺寸计算;如果当前元素
                自身没有尺寸,百分比单位参照父元素对应尺寸计算(参照父元素的属性值进行比例布局)
        3. em : 默认1em等于字体大小  16px
        4. rem : 参照文档根元素的字体大小进行尺寸设置
    2.颜色取值:
        1. rgb(255,255,255) 每一个值的取值范围是0~255  饱和度越来越高,明度越来越高
            rgb(0,0,0)  black
            rgb(255,255,255)white
        2. 英文单词
        3.rgba(r,g,b,a) a表示alpha透明度,取值0-1  (0表示透明,1表示不透明,小数表示半透明)
        4.颜色的十六进制表示
            1.语法:以#开头表示十六进制,可以使用6位字符或三位字符表示颜色,每位字符取值0-9,1-f
                (1)六位十六进制:
                    每两位为一组,代表一种三原色
                    特殊值:
                        rgb(255,0,0)       red   #ff0000
                        rgb(0,255,0)      green  #00ff00
                        rgb(255,255,255)  white  #ffffff

                (2)三位字符表示---短十六进制
                    浏览器会自动对每一位重复扩充,最终完善成6位十六进制
                        #000 --> #000000  代表黑色
                        #fff --> #ffffff  代表白色
                
                注: transparent; 表示透明的属性
                例子: border:1px solid transparent;

2.内容与边框
    1.内容
        1.尺寸属性:
            width/height : 取像素值或%
        2.内容溢出:
            元素内容超过元素自身的尺寸,称为溢出,默认溢出部分可见

            属性: overflow
            取值:
                visible  (默认溢出部分可见)
                hidden  (隐藏溢出部分)
                scroll  (窗口滚动条,强制在水平和垂直方向添加滚动条)
                auto    (自动在出现溢出的方向上添加可用的滚动条 )
    
    2.边框
        1.边框实现
            属性 : border
            作用 : 为元素统一设置上右下左四条边框
            取值 : border-width  boder-style  border-color
                1.border-width:设置边框宽度,取像素值
                2.border-style:设置边框样式,(solid,dashed虚线,dotted点线,double双线边框)
                3.border-color:设置边框颜色

                注意:
                    1.边框默认宽度为3px,border-width可以省略
                    2.边框颜色默认为黑色,boder-color也可以省略
                    3.边框样式必须写,没有默认值
        2.单边框的设置
            boder-x(top,bottom,left,right)
        
        3.实战: 网页三角标制作
            1.引入块元素(div)设置宽高为0
            2.统一设置四条边框宽度样式,透明色(transparent)
            3.想要哪个方向的三角标就设置某个方向边框颜色,显示
            注意:
                1.三角标是通过边框拼接而成,所以不能省略其他的边框
                2.行内元素不能设置宽高,自带高度(文本大小决定),空标签只能拼接上下三角标,所以一般使用块元素做三角标
    
    3.轮廓线
        类似于边框,常见于输入框获取焦点时自带的样式
            (***不占body里面的位置,占用margin的位置)
        区别:元素添加边框,在文档中是实际占位的,而轮廓线在文档中不占位
        属性:outline
        取值:color style width
        使用:取消文本输入框自带的轮廓线
    4.圆角边框
        1.作用:
            调整元素四个角的显示形状,也可以改变元素形状
        2.属性:
            border-radius
        3.取值:
            像素值或百分比,表示半径,以当前半径在元素的四个角做圆或椭圆,取一段弧,形成圆角效果
        4.取值情况:
            1.像素值 (只能去一个正圆的弧)
                border-radius:20px 表示四个角都已以20px像素做圆角效果
                border-radius:5px 10px 15px 20px (上左  上右  下右  下左)
                border-radius:5px 10px 15px
                border-radius:5px 10px   (对角相等)
            2.百分比 (根据长宽的比例画出圆半径)
                百分比参照自身宽高尺寸计算圆角半径
                根据宽高计算出两条半径,确定圆心,取
                弧,取值情况参照像素值
                border-radius:20%
        5.取值范围:
            不超过元素尺寸
            注意: 
            1.如果只有一个值,取100%等价于50%的效果
            2.单个角的取值范围是0~100%
            3.多个角统一设置,最大取50%
            4.四个角统一50%,改变元素形状(圆/椭圆,与元素尺寸有关)
    5.盒阴影
        1.属性: box-shadow
        2.取值:
            offsetX :阴影的水平偏移距离,取像素值
            offsetY :阴影垂直偏移距离
            blur : 阴影的模糊程度,取像素值,越大越模糊
            spread : (选填) 阴影的延伸距离,取像素值
            color : 阴影颜色
        3.浏览器或者元素自身都具备坐标系.默认以左上角为(0,0)点,向右向下分别为x轴和y轴的正方向

3.盒子模型
    1.一切元素皆为框
        盒模型组成:
            元素尺寸(width/height)
            内边距(padding)
            边框(border)
            外边距(margin)
        作用:各个属性互相叠加,影响元素在文档中的最终占据尺寸
        计算:默认情况下,元素最终占据尺寸为
            最终宽度 = width + 左右内边距 + 左右边框宽度 + 左右外边距
            最终高度 = height + 上下内边距 + 上下边框宽度 + 上下外边距
    2.外边距:
        1.作用:设置元素与元素之间的距离
        2.属性: margin
        3.取值:
            像素值或百分比
            1. margin:v1; (设置上右下左四个方向的外边距)
            2. margin:v1 v2;
            3. margin:v1 v2 v3;
            4. margin:v1 v2 v3 v4;
        4.特殊取值:
            1. margin:0; 取消默认外边距
            2. margin:0 auto; 设置外边距自动实现元素居中效果,auto只对左右边距有效
            3. 可以使用负值,实现元素位置的微调
        5.单方向外边距的设置
            1.属性 margin-x(top,bottom,right,left)   
            2.取值: px
        
        6.外边距合并
            1.垂直方向
                1.子元素(块元素)添加margin-top,作用于父元素上
                解决办法:
                    1.父元素添加顶部边框 border-top
                    2.父元素添加顶部内边距0.1
                        padding-top:0.1px
                    3.父元素添加overflow:hidden;
                2.垂直方向上的快元素,同时添加上下外边距,取较大的值
            2.水平方向:
                行内元素默认情况下,对盒模型的属性部分支持,水平方向的外边距会叠加显示

        7.具有默认外边距的元素
            body h1~h6 p ul ol
            取消默认外边距 margin:0
    
    3.内边距(padding)
        1.元素内容与边框之间的距离
        2.语法:
            1.padding:10px;
                设置四个方向的内边距
            2.padding:10px 20px;
                上下   左右
            3.padding:10px 20px 30px;
                左右保持一致
            4.四个值的时候分别设置(顺序为:上右下左)
        3. 单方向内边距的距离
            1.属性:
                padding-top
                padding-bottom
                padding-left
                padding-right
            2.取值:
                只取一个值
        4. 具有内边距的元素
            ul ol 表单控件 button
    
    4.box-sizing:
        1. 作用:指定盒模型的计算方式
        2. 取值:
            1.content-box 默认值 
                大部分元素遵照content-box计算在文档中的实际尺寸,计算方法:
                    盒模型各项属性依次累加
            2.border box
                表单控件默认按照border-box计算最终尺寸的
                计算方法:CSS属性width/height定义的尺寸,是border-box的尺寸,包含边框,内边距,内容
